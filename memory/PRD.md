## Creative Brief System Prompt (Overhauled 2026-04-16)
The system prompt now:
1. Extracts brand name, features, pages, colors, headline etc. from the brief message
2. Injects them directly into the prompt as `${brandName}`, `${featuresList}`, etc.
3. Requires REAL feature names (never "Feature 1, Feature 2")
4. Mandates specific section designs: navbar with max 5 links, hero with exact headline, features grid with unique SVG icons per feature, 4-column footer, dashboard with mock stats/charts/tables
5. Quality bar: 800-1100 lines, zero placeholder text
